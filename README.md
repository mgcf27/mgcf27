## 🧠 What I'm Building & Learning

### 🔧 Main Project: Assistance System
**[Assistance System](https://github.com/mgcf27/assistencesystem)** — Backend for technical assistance service orders

A backend for technical assistance service orders, built deliberately **without hiding complexity behind frameworks**:

- ✅ Custom authentication (opaque tokens, sliding sessions)
- ✅ Manual persistence with `EntityManager` (no Spring Data)
- ✅ Explicit audit logging with semantic actions
- ✅ Layered architecture + DDD influence

> ⚠️ Active development — architectural decisions are documented, not just implemented.
